# Gate Threshold Calibration

Capture at least 20 enrolled residents, five captures each, at the installed
reader. Include dark and bright-backlight captures. Run the calibration command
against the labelled probe directories and record the complete 0.30-0.60 sweep.

Choose only a threshold with FMR 0 across every impostor comparison and FNMR at
most 5%. If none exists, ship plate recognition only. Set the chosen threshold
and sharpness floor explicitly in the pilot environment and commit the report.

Recalibrate when the model version, reader position, or gate lighting changes.
